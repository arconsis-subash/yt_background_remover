# yt_background_remover
A simple local web UI for removing image backgrounds and exporting a high-quality PNG.

## Run locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the app:

```bash
python app.py
```

3. Open the browser at:

```
http://localhost:8080
```

4. Upload an image and download the transparent PNG result.
